### 项目描述
本项目为《JavaScript DOM编程艺术》一书里，最后一章案例的实现。使用原生JS写法操作DOM，实现了幻灯片动画效果，简单的AJAX与表单的提交。
### 后续工作
本项目结束后，主要的学习任务：
1. 熟悉ES6的常见语法和VUE框架。
2. 附带着学习JQuery和AJAX的更深入的知识。
3. 优化本WEBProject项目，比如可以使用框架或Jquery库，实现一个类似个人博客，使页面与用户的更多交互。
